Installing a cluster on vSphere using the Assisted Installer

You can install "Red Hat OpenShift Container Platform" on on-premise hardware or on-premise VMs by using the Assisted Installer. Installing "Red Hat OpenShift Container Platform" by using the Assisted Installer supports x86_64, AArch64, ppc64le, and s390x CPU architectures.

The Assisted Installer is a user-friendly installation solution offered on the Red Hat Hybrid Cloud Console. The Assisted Installer supports the various deployment platforms with a focus on the following infrastructures:

Bare metal

Nutanix

vSphere
