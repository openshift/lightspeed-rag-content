Viewing system event information in an Red Hat OpenShift Container Platform cluster

Events in Red Hat OpenShift Container Platform are modeled based on events that happen to API objects in an Red Hat OpenShift Container Platform cluster.
Understanding events
Events allow Red Hat OpenShift Container Platform to record information about real-world events in a resource-agnostic manner. They also allow developers and administrators to consume information about system components in a unified way.
