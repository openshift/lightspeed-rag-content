# About Operator Lifecycle Manager v1



Operator Lifecycle Manager (OLM) has been included with {product-title} 4 since its initial release. {product-title} 4.18 includes components for a next-generation iteration of OLM as a Generally Available (GA) feature, known during this phase as OLM v1. This updated framework evolves many of the concepts that have been part of previous versions of OLM and adds new capabilities.

Starting in {product-title} 4.17, documentation for OLM v1 has been moved to the following new guide:

* Extensions (OLM v1)
