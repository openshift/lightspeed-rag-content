# Installing an on-premise cluster using the Assisted Installer



You can install Red Hat OpenShift Container Platform on on-premise hardware or on-premise VMs by using the Assisted Installer for Red Hat OpenShift Container Platform.

The Assisted Installer is a user-friendly installation solution offered on the Hybrid Cloud Console. The Assisted Installer reduces installation complexity, enabling users to deploy and configure Red Hat OpenShift Container Platform clusters with minimal manual intervention.

For access to the Assisted Installer user interface and documentation, see Additional resources.
