# OpenShift Virtualization runbooks


To diagnose and resolve issues that trigger OpenShift Virtualization
alerts,
follow the procedures in the runbooks for the OpenShift Virtualization Operator. Triggered OpenShift Virtualization alerts can be viewed in the main Observe -> Alerts tab in the web console, and also in the Virtualization -> Overview tab.
Runbooks for the OpenShift Virtualization Operator are maintained in the openshift/runbooks Git repository, and you can view them on GitHub.
