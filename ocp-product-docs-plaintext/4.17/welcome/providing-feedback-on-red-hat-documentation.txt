# Providing feedback on Red Hat OpenShift Container Platform documentation



To report an error or to improve our documentation, log in to your Red Hat Jira account and submit an issue. If you do not have a Red Hat Jira account, then you will be prompted to create an account.

1. Click one of the following links:
* To create a Jira issue for Red Hat OpenShift Container Platform
* To create a Jira issue for OpenShift Virtualization
2. Enter a brief description of the issue in the Summary.
3. Provide a detailed description of the issue or enhancement in the Description. Include a URL to where the issue occurs in the documentation.
4. Click Create to create the issue.
